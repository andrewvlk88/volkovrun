# 🏃 Volkov 5K Program — 8 Weeks
## Phase 1: Base (Weeks 1-4) → 5km Continuous
## Phase 2: Speed (Weeks 5-8) → 5km PB

---

## 📐 Principles

- **3 ימי ריצה בשבוע**, לא יותר. הגוף מתחזק במנוחה.
- **Whoop Recovery קובע עומס:**
  - 🟢 ≥67% → יום קשה (אינטרוולים / טמפו)
  - 🟡 50-66% → ריצה קלה בלבד (Z2)
  - 🔴 <50% → מנוחה. לא מתפשר.
- **כלל העלייה:** לא יותר מ-10% עלייה בנפח משבוע לשבוע.
- **כלל ההתחלה:** ריצה קלה = יכול לדבר במשפטים מלאים. אם לא — יותר מדי מהר.
- **קצב אישי נוכחי:** ~9:00-10:00 דק/ק"מ (easy), ~7:30 (fast), ~8:00-8:20 (tempo)

## ⚠️ Whoop Recovery Rules

לפני כל אימון, בדוק recovery:

```
🟢 ≥67% → יום קשה (אינטרוולים / טמפו)
🟡 50-66% → המר יום קשה לריצה קלה (Z2)
🔴 <50% → מנוחה. דחה את האימון.
```

**דוגמה:** יום א' מתוכנן אינטרוולים, אבל Whoop אומר 55% 🟡 → המר לריצה קלה 4km ב-10:00 קצב. תעשה את האינטרוולים ביום ד' במקום.

## 🎯 Garmin + Whoop Integration

כל ריצה נשמרת אוטומטית:
- Garmin → נתוני ריצה (קצב, דופק, GPS, הקפות)
- Whoop → recovery לפני, strain אחרי
- volkovrun.avolkov.click → כל הנתונים מאוחדים

אחרי כל ריצה, בדוק באתר:
1. Recovery score (האם הגוף היה מוכן?)
2. דופק ממוצע (האם היה נמוך מהרגיל? = שיפור)
3. Δ Garmin vs Whoop (האם התאם?)
